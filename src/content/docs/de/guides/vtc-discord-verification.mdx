# VTC-Discord-Server-Verifizierung

Verknüpfe den Discord-Server deiner VTC mit TrucklineMP, um ihn in deinem öffentlichen Profil anzuzeigen und potenziellen Mitgliedern mehr Vertrauen zu vermitteln.

## Auf Discord-Einstellungen zugreifen

1. Navigiere zu deinem VTC-Verwaltungspanel
2. Wähle **General Settings** in der Seitenleiste aus
3. Scrolle zur **Discord-Server** Sektion

---

## Warum sollte ich meinen Discord-Server verifizieren?

- **Vertrauen aufbauen** — Zeig Bewerbern, dass dein Discord vertrauenswürdig ist
- **Engagement erhöhen** — Erleichtere es Mitgliedern, deinen Server zu finden
- **Professionelles Auftreten** — Wird gut sichtbar auf deinem VTC-Profil angezeigt
- **Community-Aufbau** — Verbinde deine TrucklineMP- und Discord-Communitys

---

## Verifizierungsprozess

### Schritt 1: Token generieren

1. Klicke auf **Generate Verification Token**
2. Ein eindeutiger Token wird für deine VTC erstellt
3. Der Token wird auf dem Bildschirm angezeigt
4. Kopiere den Token über die Kopierschaltfläche

### Schritt 2: Den Bot einladen

1. Klicke auf den Link **Invite the TrucklineMP bot**
2. Wähle deinen Discord-Server aus dem Dropdown-Menü aus
3. Gewähre dem Bot die erforderlichen Berechtigungen
4. Schließe die Einladung ab

**Erforderliche Bot-Berechtigungen:**

- Read Messages/View Channels
- Send Messages
- Use Slash Commands

### Schritt 3: Den Server verifizieren

1. Gehe zu einem beliebigen Kanal auf deinem Discord-Server
2. Gib den Befehl `/verify-server <TOKEN>` ein
3. Ersetze `<TOKEN>` durch deinen tatsächlichen Token
4. Der Bot bestätigt die Verifizierung
5. Dein Server ist nun verknüpft

### Schritt 4: Bestätigung

- Kehre zu TrucklineMP zurück
- Die Seite erkennt die Verifizierung automatisch
- Du erhältst eine Erfolgsmeldung
- Dein Discord-Server wird nun auf deinem VTC-Profil angezeigt

---

## Token-Informationen

### Token-Ablaufdatum

- Tokens laufen nach **15 Minuten** ab
- Falls dein Token abgelaufen ist, generiere einen neuen
- Jede VTC kann immer nur einen aktiven Token gleichzeitig haben

### Tokensicherheit

- Tokens sind eindeutig deiner VTC zugeordnet
- Teile Tokens nicht öffentlich
- Generiere einen neuen Token, falls dieser kompromittiert wurde

---

## Serversichtbarkeit

Lege fest, wer deinen verknüpften Discord-Server auf deinem VTC-Profil sehen kann.

### Sichtbarkeitsoptionen

**Nur du**

- Nur der VTC-Inhaber kann den Discord-Server sehen
- Für Mitglieder und die Öffentlichkeit ausgeblendet
- Nützlich während der Einrichtung oder wenn der Server privat ist

**VTC-Mitglieder**

- Für alle Mitglieder deiner VTC sichtbar
- Für die Öffentlichkeit verborgen
- Geeignet für private Communitys

**Jeder**

- Auf deinem öffentlichen VTC-Profil sichtbar
- Jeder kann deinen Discord-Server sehen
- Empfohlen für öffentliche Mitgliedergewinnung

### Sichtbarkeit ändern

1. Gehe zum Abschnitt **Discord-Server**
2. Klicke auf die gewünschte Sichtbarkeitsoption
3. Die Änderung wird automatisch gespeichert
4. Du erhältst eine Erfolgsmeldung

---

## Verknüpften Server verwalten

### Serverstatus anzeigen

- Zeige den Namen deines verknüpften Servers
- Überprüfe den Verifizierungsstatus (Verified/Unverified)
- Zeige die aktuelle Sichtbarkeitseinstellung

### Einen neuen Token generieren

- Klicke auf **Generate New Token**
- Der alte Token wird ungültig gemacht
- Verwende den neuen Token, um die Verifizierung bei Bedarf erneut durchzuführen

### Serververknüpfung aufheben

1. Klicke auf **Unlink Server**
2. Bestätige die Aktion
3. Dein Discord-Server wird aus deinem Profil entfernt
4. Du kannst später einen anderen Server verknüpfen

---

## Fehlerbehebung

### "Der Bot reagiert nicht auf meine Befehle"

- Stelle sicher, dass der Bot die Berechtigung hat, Nachrichten im Kanal zu lesen
- Überprüfe, ob du den richtigen Token verwendest
- Stelle sicher, dass sich der Bot auf deinem Server befindet
- Versuche einen anderen Kanal

### "Mein Token ist abgelaufen"

- Tokens laufen nach 15 Minuten ab
- Generiere einen neuen Token
- Verwende den neuen Token sofort

### "Die Verifizierung wird nicht in meinem Profil angezeigt"

- Aktualisiere die Seite
- Überprüfe, ob die Verifizierung erfolgreich abgeschlossen wurde
- Stelle sicher, dass die Sichtbarkeit auf "Everyone" oder "VTC Members" gesetzt ist

### "Ich möchte einen anderen Server verknüpfen"

- Hebe zuerst die Verknüpfung deines aktuellen Servers auf
- Generiere einen neuen Token
- Folge dem Verifizierungsprozess für den neuen Server

### "Der Bot hat keine Berechtigungen"

- Gehe zu den Einstellungen deines Discord-Servers
- Navigiere zu den Rollen
- Stelle sicher, dass der TrucklineMP-Bot die erforderlichen Berechtigungen besitzt
- Versuche den Befehl erneut

---

## Bewährte Vorgehensweisen

### Server einrichten

- Verwende einen dauerhaften Einladungslink
- Richte klare Kanäle und Regeln ein
- Stelle Moderatoren bereit, um neue Mitglieder willkommen zu heißen
- Halte deinen Server aktiv und ansprechend

### Sichtbarkeit

- Setze die Sichtbarkeit auf "Everyone" für öffentliche Mitgliedergewinnung
- Verwende "VTC Members" für reine Mitglieder-Communitys
- Verwende "Only You" während der Ersteinrichtung

### Wartung

- Überprüfe deinen Discord-Server regelmäßig
- Antworte zeitnah auf Fragen von Mitgliedern
- Halte deinen Server organisiert
- Aktualisiere dein VTC-Profil, wenn sich Discord ändert

---

## Voraussetzungen

Um deinen Discord-Server zu verifizieren, benötigst du:

- Die Berechtigung **Manage Server** auf dem Discord-Server
- **Verwaltungsberechtigungen** in deiner VTC auf TrucklineMP
- Einen Discord-Server, der die Richtlinien von Discord erfüllt

---

## Nächste Schritte

Nachdem dein Server verifiziert wurde, richte [Discord Linked Roles](/guides/discord-linked-roles/) ein, damit Mitglieder ihr TrucklineMP-Konto mit Discord verbinden und automatisch Rollen erhalten können.
