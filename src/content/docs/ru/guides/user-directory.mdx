# Каталог пользователей

Каталог пользователей помогает найти водителей по имени, имени пользователя или принадлежности к VTC.

## Доступ к каталогу

Откройте [trucklinemp.com/users](https://trucklinemp.com/users).

Вы можете просматривать каталог без авторизации. Некоторые детали профиля требуют авторизации или наличия в друзьях.

---

## Поиск и фильтры

Воспользуйтесь строкой поиска, чтобы сузить круг результатов:

- **Имя или имя пользователя** по частичному совпадению
- В каждом результате отображается текущий показатель VTC пользователя (если таковой имеется), когда эта информация доступна
- Пагинация для больших наборов результатов

Отдельных элементов управления фильтрацией (таких как фильтрация по VTC) нет — поиск осуществляется только по имени или идентификатору.

Нажмите на пользователя, чтобы открыть его [публичный профиль](/ru/guides/public-profile/).

---

## URL-адреса профилей

Пользователи доступны по адресам:

- `trucklinemp.com/@handle` (предпочтительно, если имя пользователя задано)
- `trucklinemp.com/user/{webId}` (запасной вариант с числовым WebID)

Имена пользователей уникальны и выбираются в [Настройках профиля](/ru/guides/profile-settings/).

---

## Конфиденциальность

Пользователи сами контролируют, что отображается в их профиле (аватар, ссылки на соцсети, подключенные стриминговые аккаунты). У заблокированных пользователей может отображаться баннер статуса. Заметки только для персонала никогда не публикуются.

---

## Жалобы на пользователей

Если вы обнаружили нарушение правил, откройте профиль пользователя и используйте **Report user**, чтобы создать [тикет в службу поддержки](/ru/guides/support-tickets/). Не беспокойте пользователей в личных сообщениях или в Discord.

---

## Доступ к API

Интеграторы могут искать пользователей через публичный API:

```
GET https://api.trucklinemp.com/users/search
GET https://api.trucklinemp.com/users/{handleOrId}
```

См. [Публичный API](/ru/guides/developers/public-api/).
